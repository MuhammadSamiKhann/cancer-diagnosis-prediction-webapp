INSTALLATION INSTRUCTIONS
=========================

Step 1: Install Required Dependencies
Open terminal/command prompt and run the following command to install all required packages:

pip install flask pandas numpy matplotlib seaborn scikit-learn scipy joblib

Step 2: Run the Project
Open terminal and run the following commands 1 by 1:

1. python train_model.py     (This trains and saves the ML models)

2. python app.py             (This starts the Flask web application)

After running app.py, open your web browser:

Dependencies Required:
- Flask: Web framework for the application
- Pandas: Data manipulation and analysis
- NumPy: Numerical computing
- Matplotlib: Data visualization
- Seaborn: Advanced data visualization
- Scikit-learn: Machine learning algorithms
- SciPy: Scientific computing
- Joblib: Model serialization and loading
